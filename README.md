## Rhodium File Locker

does what it says on the tin,

It's File locker with password based AES encryption,

how to use:
1. open app,
2. enter password(if you have encrypted something before then enter that password to decrypt, if you enter another password, it will give you corrupted output files)
3. drag-n-drop files which you want to encrypt onto the app,
4. click "Open Unencrypted Folder" button to open folder with the unencrypted files,
5. unencrypted files are only temporary and are deleted once you close the app
